## 2. 対象者・利用条件

- 本サービスは **13 歳以上** を対象とする
- **13 歳未満の利用は禁止**する。13 歳未満であることが判明した場合、データを速やかに削除する
- **13〜15 歳（GDPR 第 8 条・COPPA 対象年齢）** は保護者の同意が必要
  - 保護者のメールアドレスに確認メールを送信し、ワンクリック同意を取得する
  - 同意完了まで、ルーム入室・アバターアップロード・コイン消費を制限する

## 3. 収集する個人情報

### 3.1 アカウント登録時に収集する情報

| 情報 | 取得方法 | 備考 |
|---|---|---|
| ソーシャルプロバイダー ID（Google sub / Apple sub） | OAuth 認証 | DB に保存 |
| メールアドレス | OAuth 認証（Google / Apple から取得） | DB に保存。Apple プライベートリレーメールも含む |
| 表示名 | ユーザーが入力 | — |
| @name（ユーザー識別子） | ユーザーが入力 | 英数字・アンダースコア・3〜15 文字 |
| 年齢グループ（`young_teen` / `teen` / `adult`） | 生年月日から計算後に分類 | 生年月日は計算後に即時破棄し DB に保存しない |
| 保護者のメールアドレス | 13〜15 歳のみ。保護者同意フローで取得 | 同意確認専用。同意完了後は保持しない（要確認） |

### 3.2 サービス利用中に収集する情報

| 情報 | 取得方法 | 備考 |
|---|---|---|
| アップロードコンテンツ（VRM アバター・GLB ワールド・PNG テクスチャ） | ユーザーがアップロード | Cloudflare R2 に保存 |
| アクセサリ（GLB） | ユーザーがアップロード | 同上 |
| 購入履歴・コイン消費履歴 | 決済処理時に自動記録 | — |
| プッシュ通知トークン（APNs / FCM） | デバイスから自動取得 | 通知送信目的のみ |
| IP アドレス | API リクエスト時に自動記録（全エンドポイント） | 1 年後に自動削除 |
| User-Agent（OS・アプリバージョン） | API リクエスト時に自動記録 | 1 年後に自動削除 |

### 3.3 収集しない情報

- 生年月日（年齢計算後に即時破棄）
- 音声データ（Vivox SDK が処理する音声通話は録音・保存しない）
- 位置情報
- 決済カード番号・銀行口座情報（決済処理は Apple / Google の決済プラットフォームが処理。当社は取得しない）

| 目的 | 対象データ |

## 5. データの保持期間

| データ種別 | 保持期間 | 削除タイミング | 法的根拠 |
|---|---|---|---|
| 表示名・@name・メールアドレス・プロバイダー ID | **退会申請時に即時 NULL 化** | 申請から 30 日後に物理削除 | GDPR Art. 17（忘れられる権利） |
| アバター・ワールド・テクスチャファイル | 退会から 30 日以内 | 30 日後に物理削除 | — |
| フレンド・フォロー関係 | **退会申請時に即時削除** | — | 関係性は個人情報 |
| コイン購入履歴・消費履歴・取引キャンセル記録 | **7 年** | 7 年後に自動削除 | 日本法人税法第 126 条・GDPR Art. 6(1)(c) |
| 違反報告（受信記録）| **7 年** | 7 年後に自動削除 | 再犯・不正調査 |
| 違反報告（通報者情報）| 退会から 30 日後に匿名化 | 30 日後に NULL 化 | 通報者プライバシー保護 |
| 管理者操作ログ | **1 年** | 1 年後に自動削除 | セキュリティ監査 |
| リフレッシュトークン | 失効から 7 日後 | 自動削除 | — |
| IP アドレス・User-Agent | **1 年** | 1 年後に自動削除 | セキュリティ監査・不正検知・データ侵害時の報告先 DPA 特定（GDPR Art. 6(1)(f) 正当な利益） |
| プッシュ通知トークン | 退会時または無効化時 | 退会・無効トークン検知時に削除 | — |

## 8. セキュリティ対策

| 対策 | 内容 |
|---|---|
| 転送時の暗号化 | API 通信: TLS 1.2 以上 / ゲーム通信: DTLS 1.2 / 音声: SRTP |
| シークレット管理 | 本番環境: Google Cloud Secret Manager でキーを管理 |

## 9. Cookie・トラッキング

本サービスはネイティブモバイルアプリです。Web ブラウザ Cookie は使用しません。

アプリ内では以下のデータをローカルストレージ（Unity PlayerPrefs）に保存します：

| データ | 用途 |
|---|---|
| 認証トークン | 自動ログイン |
| 音量設定 | 設定の永続化 |
| キャッシュされたアバター・アクセサリファイル | 通信量削減 |

これらのデータはユーザーのデバイス上にのみ存在し、当社サーバーには送信されません。

## 10. アカウント削除（忘れられる権利の行使）

アプリ設定画面から1段階の確認ダイアログ経由でアカウントを削除できます。

**即時（削除申請時）:**
- 表示名・@name・メールアドレス・プロバイダー ID を即時 NULL 化
- セッション無効化（以後のログイン不可）
- 公開中のワールドを非公開に変更

**30 日後（バッチ処理）:**
- `active_users` レコードの物理削除
- アバター・ワールド・テクスチャファイルの物理削除
- @name の解放（他ユーザーが取得可能になる）

**長期保持（削除後も維持）:**
- 購入履歴・消費履歴: 7 年（税務義務。個人情報は含まない）
- 違反報告受信記録: 7 年（不正調査目的。個人情報は含まない）

## 11. 未成年者の保護

- **13 歳未満の利用は禁止**。アカウント作成時にサーバー側で年齢を検証し、13 歳未満は登録を拒否する

- 万が一 13 歳未満のデータを取得した事実が判明した場合、速やかにデータを削除する
- お問い合わせ先（nibankougen@gmail.com）に報告していただくことで対応する
